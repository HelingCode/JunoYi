# 📌 JunoYi-Uniapp 项目说明

JunoYi-Uniapp为 JunoYi 后端框架配套的通用小程序模版项目。
由于前后端工程结构、技术栈及职责不同，为了保持代码清晰与仓库职责单一，前端项目已独立拆分至单独仓库进行维护。
